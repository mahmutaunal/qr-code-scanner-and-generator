# QR Code Scanner & Generator - Android App

<img src="/readme/logo.png" align="left"

QR Code Scanner & Generator is an QR code or barcode scanner and generator.
QR Code Scanner & Generator is free.
No registration is required to use the QR Code Scanner & Generator application.

QR Code Scanner & Generator is available on the Google Play Store.

## About

With this application, you can read QR code or barcode. You can create QR code and additionally share the QR code you have created.

## Features

The android app lets you:
- QR code or barcode reader
- QR code generator
